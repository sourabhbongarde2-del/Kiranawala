Hum kya jaankari lete hain

Jab aap order karte ho, hum ye jaankari lete hain taaki aapka order sahi se deliver ho sake:

  • Naam aur mobile number
  • Delivery address / PIN code
  • Order details (kya kharida, kitni quantity, price)
  • Payment mode (COD ya UPI) — lekin hum aapke bank/UPI PIN jaise sensitive details kabhi store nahi karte

Is jaankari ka istemaal kaise hota hai

  • Aapka order process aur deliver karne ke liye
  • Aapse order ke baare mein contact karne ke liye (confirmation, delivery updates)
  • Agar aapne notification permission di hai, to naye offers ke baare mein batane ke liye

Data kahan store hota hai

Aapki jaankari Google Firebase (Firestore) par securely store hoti hai. Sirf store ka malik apne customers ke orders dekh sakta hai — koi aur customer ye jaankari nahi dekh sakta.

Aapke rights

  • Aap kabhi bhi apna data delete karne ki request kar sakte ho — store ke WhatsApp/phone number par contact karo
  • Aap notification permission kabhi bhi apne browser settings se band kar sakte ho

Sawal ho to

Kisi bhi privacy-related sawal ke liye, store se seedha contact karo (contact details store page par milenge).
